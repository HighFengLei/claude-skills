# frontend-design

**来源**: anthropics/skills  
**许可**: Apache 2.0 (详见 LICENSE.txt)

## 作用

指导 Claude 创建高质量、有设计感的前端界面，避免千篇一律的"AI 审美"喵。

## 适用场景

- 构建网页、落地页、仪表盘、React 组件、HTML/CSS 布局等前端界面喵
- 对现有界面进行美化和风格改进喵
- 需要独特的视觉效果（大胆的字体、配色、动效、布局）喵

## 核心要点

- 强调**有意图的设计**：选择一个清晰的美学方向（极简/极繁/复古/自然等）并贯彻到底喵
- 避免通用字体（Inter、Roboto、Arial）和俗套配色（紫色渐变+白底）喵
- 注重排版、色彩搭配、动效、空间构图和背景细节喵
