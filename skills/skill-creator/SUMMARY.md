# skill-creator

**来源**: anthropics/skills  
**许可**: Apache 2.0 (详见 LICENSE.txt)

## 作用

创建、修改和优化 skill。包含完整的 skill 开发工作流和评估框架喵。

## 适用场景

- 想为 Claude 创建自定义 skill 时喵
- 想要改进现有 skill 的性能和质量喵
- 需要对 skill 进行基准测试和评估喵

## 核心要点

- 完整的 skill 创建流程：构思 → 草稿 → 测试 → 评估 → 迭代喵
- 包含评估框架（eval-viewer），支持定量和定性分析喵
- 可扩展测试集进行大规模验证喵
